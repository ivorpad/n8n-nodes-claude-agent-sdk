# CLAUDE.md

Purpose: this repository contains `n8n-nodes-claude-agent-sdk`, an n8n
community-node package for self-hosted n8n installations that want to run
Claude Agent SDK / Claude Code tasks inside workflows.

Scope: package source, credentials, channel adapters, session-memory nodes,
tests, public user documentation, sanitized examples, and necessary
build/release metadata. Deployment stacks, private operational notes, generated
outputs, caches, local agent config, packed tarballs, logs, and secrets do not
belong here.

See `AGENTS.md` for contributor and agent instructions.
