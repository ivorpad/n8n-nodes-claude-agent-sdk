# CLAUDE.md

Purpose: this repository is the public open-source home for
`n8n-nodes-claude-agent-sdk`, an n8n community node package that integrates the
Claude Agent SDK with self-hosted n8n workflows.

Publication scope: keep this repository limited to source code, tests, public
documentation, sanitized examples, and necessary build/release metadata. Local
artifacts, private deployment details, generated outputs, scrape caches,
personal agent config, packed tarballs, logs, and secrets do not belong here.

See `AGENTS.md` for contributor and agent instructions.
